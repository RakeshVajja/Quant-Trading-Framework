# Quant-Trading-Framework
Modular algorithmic trading framework with backtesting and risk engine

## Features
- EMA crossover strategy
- Backtesting engine
- Performance metrics (Drawdown, Expectancy, Win rate)
- Risk management engine (coming soon)

## Architecture
- Data Engine
- Strategy Engine
- Risk Engine
- Backtester
- Metrics Module
